Guía para un Diseño Web Responsivo

Aprende a crear sitios que se adapten a cualquier pantalla.

¿Qué es el Diseño Responsivo?

Es la práctica de crear sitios web que ofrecen una experiencia de usuario óptima en una amplia gama de dispositivos, desde computadoras de escritorio hasta teléfonos móviles y tabletas. Se trata de que el contenido se vea y funcione bien sin importar el tamaño de la pantalla.

Principios Clave

1. El Viewport

La base de todo. Con la etiqueta `<meta name="viewport">`, le decimos al navegador que el ancho de la página debe ser igual al ancho del dispositivo. Sin esto, el diseño responsivo simplemente no funciona.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

2. Unidades Relativas

El uso de unidades como `%`, `em`, `rem`, y `vw` es fundamental. A diferencia de los píxeles, estas unidades se ajustan según el tamaño del contenedor o del viewport, garantizando que los elementos escalen de forma fluida.

font-size: 1.2rem;
width: 80%;
margin: 2vw;

3. Media Queries

Las Media Queries nos permiten aplicar diferentes estilos CSS en función de las características del dispositivo, como su ancho. Esto nos da el control para reorganizar el diseño en diferentes "puntos de ruptura" (breakpoints).

@media (min-width: 768px) {
  .grid-container {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
  }
}
